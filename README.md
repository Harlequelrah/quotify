# **`QUOTIFY`**

# Description

Ceci est un projet backend d'API de gestion des citations .

## Fonctionnalités

- Creation d'auteurs

- CRUD de citations

- CRUD de tags

- authentification

- Enregistrement de logs
## Installation

- **Avec Github :**
  ```bash
  git clone https://github.com/Harlequelrah/Quotify
  ```

## Utilisation

- **Se rendre dans le repertoire du projet** :

```bash
  cd monprojet
```

-  **Création d'un environnement virtuel :**
```bash
  python -m venv env
```
ou
```bash
  virtualenv env
```

-  **Installation des dépéendances :**
```bash
  pip install -r requirements.txt
```

-  **Initialisation des variables d'environnement :**
```bash
  cp .env.example .env
```

-  **Initialisation d'alembic (Optionnel) :**
```python
  alembic init alembic
```


- **Lancement du serveur :**
```python
  elrahapi run
```

## `Technologies et bibliothèques`

- Framework ElrahAPI
- Python
- Pydantic
- FASTAPI
- SQLAlchemy
